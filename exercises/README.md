# Exercises — Capstone Project

This directory contains starter files for the capstone project in Chapter 10.

## What to build

A CLI tool called `wordcount` that counts lines, words, and characters in a file. The full requirements are detailed in [../chapters/10-capstone.md](../chapters/10-capstone.md).

Follow the steps there — they walk you through every command you need. This directory is intentionally left minimal so you do the scaffolding yourself (it's part of the learning).
